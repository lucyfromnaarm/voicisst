Welcome

Voicisst turns your speech into clear, polished writing — in any app, wherever your cursor is.

Private by design: everything runs on your own computer. Your voice never leaves it.

The next steps check your microphone, hotkey, and helpers. Every step is optional — use Skip freely. You can run this setup again any time from Help.

Permissions

Linux

Two one-time permissions let Voicisst see your hotkey and type for you:

  1. Join the input group so the hotkey works everywhere:
    sudo usermod -aG input $USER
    Then log out and back in once.
  2. Start the typing helper:
    systemctl --user enable --now ydotoold

macOS

macOS asks you to approve three things, once each. Open System Settings → Privacy & Security, then add the app you run Voicisst from (usually your terminal) to:

  1. Accessibility — lets Voicisst type into other apps.
  2. Input Monitoring — lets the hotkey work everywhere.
  3. Microphone — lets Voicisst hear you.

Windows

Nothing special is needed on Windows. You can move on.

    Quick fixes

    • The hotkey does nothing. Make sure dictation is running (voicisst run). On Linux, you may need to join the input group — see Setup → Permissions.
    • My words come out empty or wrong. Test the microphone in Setup — it may be muted or very quiet.
    • Polish is slow. Pick a smaller polish model in Settings, or turn polish off — you still get the raw transcript.
    • The dashboard says “Not running”. Start dictation with voicisst run --ui.
